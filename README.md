# Conversor-Decimal-Binario
Una simple herramienta que convierte números decimales a binarios en diferentes sistemas de representación.

- Binario positivo.
- Signo y magnitud.
- Compemento a la base menos uno.
- Complemento a la base.
- Exceso a dos.
